== Description ==

该插件支持使用新浪云存储SCS作为附件存储空间。
This is a plugin that use Sina Cloud Storage(SCS) for attachments remote saving.

支持xmlrpc。
Support xmlrpc.

支持自定义上传路径。
Custom upload url supported.

详见http://blog.gimhoy.com/archives/scs-support.html.
For more details, please visit http://blog.gimhoy.com/archives/scs-support.html.

==  Installation ==

1. 下载，解压缩并上传到WordPress插件目录
2. 在插件管理后台激活插件
3. 工具 > SCS Setting 

1. Download, unzip and upload to your WordPress plugins directory
2. activate the plugin within you WordPress Administration Backend
3. Go to Tools > SCS Setting
